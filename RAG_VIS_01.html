🎯 핵심 인사이트

1. 지식 자동화가 주목적

기업들은 단순 생성보다 문서 기반의 정확한 정보 활용과 자동화에 관심을 두고 있습니다. 사내 지식 검색과 문서 요약이 49%로 1위를 차지했습니다.

2. 맞춤형 구축이 대세

47%가 RAG 기반 맞춤형 AI 시스템 구축을 선호합니다. 단순 API 호출이나 자체 모델 개발보다 자사 데이터를 활용하는 현실적 접근을 선호합니다.

3. 데이터가 가장 큰 장벽

기술보다 데이터 준비가 실질적인 허들입니다. 39%가 데이터가 흩어져 있고 구조화되어 있지 않다고 답했으며, 데이터 구조화를 최우선 고려사항으로 꼽았습니다.

4. 보안과 운영 환경이 핵심

실제 운영을 위해서는 데이터 보안 및 권한 체계(50%), 운영 및 관리 자동화(48%)가 필수적입니다. 모델 성능보다 안정적이고 지속 가능한 운영 기반이 중요합니다.

5. 아직은 초기 단계

실제 도입은 11%에 불과하며, 31%가 내부 검토 중, 29%는 가능성만 열어둔 상태입니다. RAG 인지도는 높지만 실무 적용은 아직 초입 단계입니다.
